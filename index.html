CIS 19x: Android

Overview Lectures Assignments Staff

This course provides an introduction to Android application development. The emphasis would be on using different native and 3rd party Java libraries to write Android applications. This course is most useful for students who have some programming experience and want to learn effective techniques to build complex Android applications.

Sample topics covered include:
  • Layouts, animation, and Material Design principles
  • Threads, client-server interaction and API consumption
  • Integration with Android system services
  • Modern Android libraries (Retrofit, Butter Knife, etc.)
  • Storage and preferences management
  • App architecture and design patterns
  • Hardware integration (camera, GPS, etc.)

Prerequisites

CIS 120 or Java programming experience
